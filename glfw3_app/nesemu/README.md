nesemu
=========

NES Emulator application

## nesemu の概要   

・NES ファイルを実行するアプリケーション   
   
#### 操作方法

F1:  ファイラーの On/Off   
F2:  制御盤（ステートの切り替え、セーブ、ロード、マスターボリューム）   
F5:  ログ表示と簡単なデバッグ機構
  Status: status   
  disasm: xxxxL   
  read:   xxxx   
  write:  xxxx:yy zz ...   
   
1:   SELECT   
2:   START   
Z:   A   
X:   B   
   
↑:  Dir UP   
↓:  Dir Down   
→:  Dir Right   
←:  Dir Left
